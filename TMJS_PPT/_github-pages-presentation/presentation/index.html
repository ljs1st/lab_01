기출 유형은 달라도 제시문 이해, 기준 설정, 내 경험 연결이 답변의 중심이 됩니다

제시문 핵심+쟁점 기준+내 입장+경험 연결
  • 자사고형: 고사나 역사적 상황을 바탕으로 가치 판단을 묻습니다
  • 국제고형: 철학·문학·역사 제시문을 연결해 기준을 세우게 합니다
  • 외고형: 비유적 제시문을 읽고 문화·사회 문제로 확장하게 합니다

INDIVIDUAL QUESTION

개별문항은 학생부와 자소서에서 내가 설명해야 할 빈칸을 묻습니다

활동명, 독서명, 학습법, 진로계획을 썼다면 왜, 어떻게, 무엇을 배웠는지 준비해야 합니다

생활기록부 기반

  • 대회·동아리·토론 활동의 실제 역할
  • 독서 활동의 선택 이유
  • 세특에 드러난 개념 이해

자기소개서 기반

  • 학습법의 장단점
  • 지원동기의 진정성
  • 동아리·탐구 활동의 구체 과정

꼬리 질문

  • 왜 그렇게 생각했나
  • 다른 선택지는 없었나
  • 입학 후 어떻게 이어갈 것인가

ANSWER STRUCTURE

답변은 길게 말하기보다 결론과 근거가 보이게 말해야 합니다

첫 문장에서 내 생각을 말하고, 뒤 문장에서 근거와 경험을 차례로 붙이면 훨씬 선명해집니다

PREP

Point → Reason → Example → Point. 의견형 질문에 가장 안정적

경험형

계기 → 과정 → 결과 → 의미. 서류 기반 질문에 적합

제시문형

요약 → 쟁점 → 입장 → 경험. 공통문항에서 강합니다

탐구형

문제 → 방법 → 발견 → 확장. 과학·교과 개념 질문에 적합

INTERVIEW ATTITUDE

말의 내용만큼 시선, 속도, 자세도 답변의 인상을 만듭니다

좋은 답변도 너무 빠르거나 시선이 흔들리면 전달력이 떨어집니다. 말하는 태도까지 연습해야 합니다

상황해야 할 태도주의할 점
질문 청취끝까지 듣고 핵심어를 잡음중간에 끼어들지 않기
답변 시작결론부터 짧게 제시단답형으로 끝내지 않기
시선 처리면접관의 이마·입가 주변 응시두리번거리거나 고개 숙이지 않기
모르는 질문정중히 재질문 또는 한계를 인정꾸며내서 말하지 않기

FINAL ACTION PLAN

오늘 이후 할 일은 세 가지입니다: 학교 좁히기, 소재 고르기, 답변 만들기

막연하게 학교 이름만 고르지 말고, 학교 선택과 자소서 소재, 면접 답변을 한 장의 준비표로 묶어봅니다

1. 학교 후보 3개

도전·적정·안정으로 나누고 교육과정과 전형 방식을 비교합니다.

2. 핵심 소재 5개

학습, 진로, 탐구, 독서, 인성 소재를 면접 질문으로 바꿉니다.

3. 답변 구조 10개

공통문항과 개별문항을 예상하고 60초 답변으로 훈련합니다.

합격 전략은 “좋은 학생”을 증명하는 것이 아니라 “이 학교에 맞는 학생”을 설득하는 것입니다.
